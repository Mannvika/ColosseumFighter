# Working Title: Colosseum Fighter

# Description
2D top-down 1v1 multiplayer fighting game with twin stick controllers. 

# MVP Goals
- Help re-familiarize with Unity
- Apply game programming patterns (state machines, decoupled control systems, composition, inheritance, interfaces, etc.)
- Understand game design principles (game loop, character balance)
- Understand game networking principles
- Implement a rollback feature for client-side prediction
